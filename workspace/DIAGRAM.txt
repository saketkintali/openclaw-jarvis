  Text message reply:
    Claude → response text → OpenClaw → WhatsApp

  Voice message reply (user sends audio):
    check_audio.py → transcribe → get_ai_response(transcript) → Claude
      → send_whatsapp_audio() → edge-tts → OGG → WhatsApp
      → (fallback to text if TTS fails)

  Audio-on-demand ("tell me a joke aloud"):
    check_speak.py strips "aloud" → get_ai_response("tell me a joke")
      → Claude answers → send_whatsapp_audio() → edge-tts → OGG → WhatsApp
    Claude main session: sees "aloud" → replies NO_REPLY (silenced)
